# Запрос на создание Waitlist Landing для Confly

**Контекст задачи:**
Создание wait-landing страницы для стартапа Confly (тренажёр публичных выступлений).

**Ключевые фичи Confly:**
- Загрузка презентации + фото любого человека (инвестор, профессор, жюри). ИИ оживляет его в говорящий аватар.
- Выступление вживую — аватар смотрит слайды и задаёт острые вопросы.
- После — скоркарта и фидбек.
- Сценарии: стартап-питч, Demo Day, защита диплома, хакатон, школьная презентация, корпоративный борд.
- Главная киллер-фича: живой аватар из любого фото.

**Требования к лендингу:**
- Показ количества зарегистрированных юзеров.
- Показ места в очереди (например, #34).
- Таймер до открытия демо.
- Дизайн: черный фон, желтые акценты + гармоничные цвета.

**Текущий статус:**
Разработка лендинга завершена (режим ACT).

**Что было сделано:**
1. Инициализирован проект на Vite + React + TypeScript в папке `landing`.
2. Установлен и настроен Tailwind CSS 4-й версии, определена кастомная цветовая тема (основной цвет - черный `#0A0A0A`, акцентный - желтый `#FBBF24`). Установлена библиотека иконок `lucide-react`.
3. Реализован главный компонент `App.tsx`:
   - Навигационная панель.
   - Hero-секция с основным УТП и формой регистрации в Waitlist.
   - Имитация отправки формы с показом "Вы в списке!" и случайным местом в очереди (в диапазоне 34-133).
   - Секция со счетчиком (1,284 пользователя) и таймером до открытия демо (имитация).
   - Секция "Киллер-фича" с красивым UI-мокапом (имитация звонка с живым аватаром и твоей камерой).
   - Секция со сценариями (сетка из 6 карточек).
   - Подвал (Footer).
4. Проект запущен и доступен локально.

**Использованные технологии:** Vite, React, TypeScript, Tailwind CSS, Lucide React.
